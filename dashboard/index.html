Scanner API

Endpunkt

http://76.13.138.73:8000


GET /health — Systemstatus
POST /scan — Scan starten
GET /status/{scan_id} — Status
GET /dossier/{scan_id} — PDF herunterladen
GET /dossier/latest — Letztes PDF

Schnelltest

Scan starten:

curl -X POST http://76.13.138.73:8000/scan \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'

Status prüfen:

curl http://76.13.138.73:8000/status/<scan_id>

PDF downloaden:

curl http://76.13.138.73:8000/dossier/latest -o dossier.pdf

CMD API
